# 🚀 MyTools - 快速开始

## 一分钟启动指南

### 第一步：安装依赖

```bash
# 推荐使用 pnpm（更快、更节省空间）
pnpm install

# 或使用 npm
npm install
```

### 第二步：启动开发服务器

```bash
pnpm dev
```

就这么简单！🎉 浏览器会自动打开 `http://localhost:3000`

## 主要功能

### 🌙 主题切换
点击右上角图标即可切换明暗主题，自动保存偏好。

### 📋 一键复制
所有工具都支持一键复制结果到剪贴板。

### 🔒 隐私优先
所有处理都在浏览器本地完成，不向服务器发送任何数据。

### 📱 响应式设计
完美支持手机、平板和桌面。

## 添加新工具（3步）

### 1. 添加工具配置
在 `src/lib/tools-data.ts`：
```typescript
myNewTool: {
  label: "我的新工具",
  description: "工具描述",
  badges: ["标签"],
  category: "分类",
  path: "/tools/my-new-tool",
  icon: "Icon",
}
```

### 2. 创建工具页面
在 `src/views/tools/MyNewToolView.vue`：
```vue
<template>
  <div class="container mx-auto max-w-4xl py-10 px-4">
    <!-- 工具界面 -->
  </div>
</template>

<script setup lang="ts">
// 工具逻辑
</script>
```

### 3. 添加路由
在 `src/router/index.ts`：
```typescript
{
  path: '/tools/my-new-tool',
  name: 'my-new-tool',
  component: () => import('../views/tools/MyNewToolView.vue'),
}
```

完成！🎉
