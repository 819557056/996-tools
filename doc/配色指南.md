# 🎨 配色指南

## 为什么样式有差异？

### 原因说明
1. **原网站** ([101webtools.com](https://101webtools.com/))
   - 使用 Next.js + React + **shadcn/ui** 组件库
   - shadcn/ui 有完整的设计系统和组件样式
   - 包含特定的阴影、动画、圆角等细节

2. **重构版本**（本项目）
   - 使用 Vue 3 + 自定义组件
   - 从零开始构建组件，功能完全相同
   - 初始使用通用配色，而非原网站的品牌色

## 🎨 如何自定义配色

### 方法1: 修改 CSS 变量

编辑 `src/assets/main.css` 文件中的颜色值：

```css
:root {
  /* 主色调 - 修改这个来改变整体色调 */
  --primary: 221.2 83.2% 53.3%;  /* HSL格式: 色相 饱和度% 亮度% */
  
  /* 示例：改为绿色主题 */
  --primary: 142 76% 36%;  /* 绿色 */
  
  /* 示例：改为紫色主题 */
  --primary: 262 83% 58%;  /* 紫色 */
  
  /* 示例：改为橙色主题 */
  --primary: 25 95% 53%;  /* 橙色 */
}
```

### 方法2: 使用在线工具生成配色

推荐工具：
- [Realtime Colors](https://realtimecolors.com/) - 实时预览配色
- [shadcn/ui Themes](https://ui.shadcn.com/themes) - shadcn/ui 主题生成器
- [Coolors](https://coolors.co/) - 配色方案生成

### 方法3: 复制原网站的配色

如果您想**完全匹配**原网站，可以：

1. 打开 [101webtools.com](https://101webtools.com/)
2. 按 `F12` 打开开发者工具
3. 在 Elements 标签中查看 `:root` 的 CSS 变量
4. 复制到 `src/assets/main.css`

## 🎯 其他视觉差异

除了配色，还有这些可能的差异：

### 1. 阴影效果
原网站可能使用了更细腻的阴影：

```css
/* 在组件中添加 */
.card {
  box-shadow: 0 1px 3px 0 rgba(0, 0, 0, 0.1), 0 1px 2px 0 rgba(0, 0, 0, 0.06);
}
```

### 2. 圆角
调整 `--radius` 变量：

```css
--radius: 0.5rem;  /* 8px - 当前 */
--radius: 0.375rem;  /* 6px - 更锐利 */
--radius: 0.75rem;  /* 12px - 更圆润 */
```

### 3. 字体
如果想使用特定字体，在 `index.html` 添加：

```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
```

然后在 `tailwind.config.js` 中：

```js
theme: {
  extend: {
    fontFamily: {
      sans: ['Inter', 'sans-serif'],
    },
  },
}
```

### 4. 动画和过渡
原网站可能有更多的微交互动画，可以在组件中添加：

```css
transition: all 0.2s ease-in-out;
```

---

**提示**: HSL 格式说明
- **H (Hue)**: 色相 0-360°（0=红, 120=绿, 240=蓝）
- **S (Saturation)**: 饱和度 0-100%（0=灰, 100=纯色）
- **L (Lightness)**: 亮度 0-100%（0=黑, 50=正常, 100=白）
