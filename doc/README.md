## 🎯 核心概念

### 总站（Portal）
- 作为统一入口，聚合展示所有分站
- 提供全局搜索和导航
- 展示文章和教程
- 域名：`996tools.com`

### 分站（Site）
- 垂直领域的专业工具集合
- 独立部署和维护
- 域名：`{slug}.996tools.com`
- 示例：`crypto.996tools.com`（国密工具站）

### 工具（Tool）
- 具体的在线工具功能
- 属于某个分站
- 路径：`/tools/{tool-key}`

### 内容（Content）
- 文章、教程、文档
- 可以属于总站或某个分站
- 路径：`/articles/{slug}`
